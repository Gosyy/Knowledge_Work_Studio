# Product Spec — KW Studio

## Summary

KW Studio is an AI workspace for knowledge work.
Users upload files, describe tasks in natural language, and receive finished work products rather than only text answers.

### MVP workflows
- DOCX → edited DOCX
- PDF → summary/report
- notes + materials → PPTX presentation
- CSV/XLSX → analysis + charts + downloadable outputs

## MVP scope
Included:
- chat workspace
- file upload and storage
- DOCX workflows
- PDF workflows
- slide generation
- spreadsheet and data analysis
- Python execution
- artifact output and download

Out of scope:
- full autonomous web operator
- realtime collaboration
- broad file-format zoo
- general-purpose app builder UI
